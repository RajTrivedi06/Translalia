# Agent Temporary Documentation

## What this folder is for

Agent-generated temporary documentation, sanity reports, and investigation outputs **for human review**.

## Rules for agents

- **Only** place temp docs, sanity reports, and review materials in this folder.
- Do **not** put temporary or WIP content elsewhere in `docs/` or at repo root.
- Files here are short-lived; the maintainer will clean up or archive as needed.
- Use descriptive filenames (e.g. `2026-02-24-sanity-report.md`, `investigation-xyz.md`).

## What belongs here

- Sanity reports
- Investigation summaries
- Temporary analysis or findings
- Draft reports awaiting review

## What does not belong here

- Long-term project documentation (use the canonical `docs/` structure)
- Code or config changes
- Secrets or credentials
